# ⏱️ Stopwatch Web Application

A simple and interactive stopwatch application built using HTML, CSS and JavaScript.

---

## 🚀 Features

- ▶️ Start timer
- ⏹️ Stop timer
- 🔄 Reset timer
- ⏱️ Real-time stopwatch display
- 💡 Beginner-friendly project
